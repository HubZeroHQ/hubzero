# Archive — Obsolete Marketing Website Direction

**Status: Archived 2026-07-12.** The public marketing website direction documented here — including homepage composition, services/work/about/contact pages, navigation, hero, section ordering, component compositions, marketing copy structure, visual rhythm, layout patterns, motion concepts, card designs, and CTA patterns — has been reset.

This is not an iteration or a refinement of the old direction. It is a full design reset for the public marketing website.

## What this means

- Everything in this folder is preserved **for historical reference only**.
- Nothing in this folder should be treated as implementation guidance for new marketing-website work.
- Do not carry forward the old homepage structure, page structures, component compositions, or design language into the new direction.
- The new redesign lives in [`DESIGN/NEXT`](../DESIGN/NEXT) and starts from a blank slate.

## What was archived

- `ARCHIVE/ARCHITECTURE/` — the marketing-site-facing architecture documents (product vision/positioning, brand strategy, information architecture, user journeys, content strategy, page specifications, the v2 design system, SEO strategy, implementation roadmap, homepage design, responsive design standards, company four-pillar structure as expressed through site navigation, feature specification).
- `ARCHIVE/DESIGN/` — the `V3` and `V4` marketing-site design explorations and the AI design guide that governed how those pages were composed.
- `ARCHIVE/docs/design-reviews/` — design reviews of the now-archived marketing site.
